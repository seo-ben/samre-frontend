# Liste Complète des Pages — SAMRE
## Contenu détaillé de chaque page

---

## A. PAGES COMMUNES (Tous utilisateurs)

---

### A1. Splash Screen
- Logo SAMRE centré
- Animation de chargement
- Redirection automatique vers sélection de langue

---

### A2. Sélection de la Langue
- Logo SAMRE en haut
- 3 boutons de sélection : **Français / English / Português**
- Mémorisation du choix pour les prochains lancements

---

### A3. Pages Publicitaires (x3)
- Image publicitaire plein écran
- Indicateur de progression (points 1/2/3)
- Bouton **"Passer"** en haut à droite
- Défilement automatique ou manuel

---

### A4. Sélection du Pays
- Logo SAMRE en haut
- Liste déroulante des pays éligibles *(Togo en premier)*
- Mention **"Pays bientôt disponible"** pour les pays non encore actifs
- Indicatif téléphonique qui s'adapte automatiquement
- Bouton **"Continuer"**

---

### A5. Page d'Inscription
- Champ numéro de téléphone *(avec indicatif du pays sélectionné)*
- Bouton **"Recevoir le code OTP"**
- Lien vers la page de connexion

---

### A6. Vérification OTP
- Texte : *"Code envoyé sur WhatsApp au +XXX"*
- Champ de saisie du code OTP (6 chiffres)
- Minuterie de validité du code
- Bouton **"Valider"**
- Lien **"Renvoyer le code"**

---

### A7. Vérification Email *(si email renseigné)*
- Texte explicatif
- Champ de saisie du code reçu par email
- Bouton **"Valider"**
- Lien **"Renvoyer l'email"**

---

### A8. Choix du Type de Compte
- Titre : *"Vous êtes ?"*
- 3 cartes de sélection :
  - 👤 **Candidat** *(étudiant, chercheur de stage)*
  - 🏢 **Entreprise** *(recruteur, organisation)*
  - 🔘 **Rien des deux** *(visiteur, partenaire)*
- Bouton **"Continuer"**

---

### A9. Page de Connexion
- Champ numéro de téléphone
- Bouton **"Recevoir le code OTP"**
- Lien **"Créer un compte"**

---

### A10. Notifications — Centre de Notifications
- Liste chronologique des notifications
- Icône + titre + description courte + date/heure
- Badge de nombre de notifications non lues
- Bouton **"Tout marquer comme lu"**
- Types affichés :
  - Nouvelle offre
  - Réponse à candidature
  - Nouvel événement
  - Rendez-vous programmé
  - Expiration abonnement
  - Mise à jour profil

---

### A11. Paramètres
- **Langue** : changer la langue de l'interface
- **Pays** : changer de pays
- **Notifications** :
  - Activer / désactiver Push
  - Activer / désactiver Email
  - Activer SMS *(avec frais — in-app uniquement)*
  - Activer Appel *(avec frais — in-app uniquement)*
- **Compte** :
  - Modifier le mot de passe / code
  - Supprimer le compte *(avec confirmation)*
- **À propos** : version de l'application

---

## B. PAGES CANDIDAT

---

### B1. Complétion du Profil Candidat
- **Informations personnelles :**
  - Nom, Prénom
  - Photo de profil *(upload obligatoire)*
  - 2 à 3 photos plein pied *(upload obligatoire)*
  - Date de naissance
  - Téléphone, Email
  - Pays, Région, Préfecture, Commune, Ville
  - Moyen de déplacement *(oui / non)*
- Barre de progression du profil *(ex: 60% complété)*
- Bouton **"Enregistrer"**
- Avertissement si profil incomplet *(blocage candidature)*

---

### B2. Mes Compétences
- Liste des compétences ajoutées *(nom + niveau)*
- Niveaux : Débutant / Intermédiaire / Avancé / Expert
- Bouton **"Ajouter une compétence"**
- Actions par compétence : Modifier / Supprimer

---

### B3. Mes Formations
- Liste des formations ajoutées
- Chaque formation affiche : établissement, diplôme, dates
- Bouton **"Ajouter une formation"**
- Formulaire d'ajout :
  - Nom de la formation
  - Établissement
  - Diplôme
  - Date début / Date fin
  - Description
- Actions : Modifier / Supprimer

---

### B4. Mes Expériences Professionnelles
- Liste des expériences ajoutées
- Chaque expérience affiche : poste, organisation, dates
- Bouton **"Ajouter une expérience"**
- Formulaire d'ajout :
  - Poste
  - Organisation
  - Date début / Date fin
  - Description
- Actions : Modifier / Supprimer

---

### B5. Page d'Accueil Candidat — Liste des Offres
- Barre de recherche
- Filtres : zone géographique / catégorie / date
- Liste des offres disponibles, chaque offre affiche :
  - Image
  - Titre du poste
  - Nom de l'entreprise
  - Description courte
  - Date limite
  - Badge *"Nouvelle"* si récente
- Indicateur de quota restant *(ex: "1 candidature gratuite cette semaine")*

---

### B6. Détail d'une Offre
- Image de l'offre
- Titre du poste
- Nom et logo de l'entreprise
- Description complète
- Missions
- Profil recherché
- Conditions
- Date limite
- Bouton **"Postuler"** *(vérifie profil + quota + wallet)*
- Bouton **"Enregistrer"**
- Bouton **"Partager"**

---

### B7. Mes Candidatures
- Liste de toutes les candidatures soumises
- Chaque candidature affiche :
  - Titre de l'offre
  - Nom de l'entreprise
  - Date de soumission
  - Statut coloré : *Soumise / En cours / Acceptée / Refusée*
- Filtre par statut
- Tap sur une candidature → détail de l'offre

---

### B8. Page des Événements — Candidat
- Barre de recherche
- Filtres : date / catégorie / lieu
- Liste des événements, chaque événement affiche :
  - Image
  - Titre
  - Date et heure
  - Lieu
  - Organisateur
- Bouton **"Créer un événement"**

---

### B9. Détail d'un Événement
- Image de l'événement
- Titre
- Date, heure, lieu
- Organisateur
- Description complète
- Programme
- Contact
- Bouton **"Participer"**
- Bouton **"Partager"**

---

### B10. Créer un Événement
- Champs :
  - Titre
  - Image
  - Catégorie *(liste définie par l'admin)*
  - Date et heure
  - Lieu
  - Description
  - Programme
  - Contact
  - Type d'action attendu : Participer / Sponsoriser / Financer
- Bouton **"Publier"**

---

### B11. Wallet & Paiement — Candidat
- Solde actuel du wallet
- Bouton **"Recharger"** *(montant minimum 500 FCFA)*
- Abonnements disponibles *(définis par l'admin)* :
  - Nom du plan
  - Durée
  - Tarif
  - Accès débloqués
  - Bouton **"Souscrire"**
- Plan actif affiché *(avec date d'expiration)*
- Historique des transactions :
  - Date
  - Type *(recharge / candidature / abonnement)*
  - Montant

---

### B12. Mon Profil Candidat — Vue Publique
- Photo de profil
- Nom, Prénom
- Ville / Pays
- Compétences
- Formations
- Expériences
- Photos plein pied *(selon règles de visibilité définies par l'admin)*
- Badge de vérification *(si validé par l'admin)*
- Bouton **"Modifier mon profil"**
- Bouton **"Me déclarer embauché"**
- Bouton **"Supprimer mon compte"**

---

## C. PAGES ENTREPRISE

---

### C1. Complétion du Profil Entreprise
- Nom de l'entreprise
- Logo / Photo
- Secteur d'activité
- Description
- Téléphone, Email
- Pays, Région, Préfecture, Commune, Ville
- Géolocalisation du lieu de travail

---

### C2. Tableau de Bord Entreprise
- Nombre d'offres publiées
- Nombre de candidatures reçues
- Nombre de rendez-vous programmés
- Raccourcis rapides :
  - Publier une offre
  - Voir les candidatures
  - Parcourir les candidats

---

### C3. Publier une Offre
- Champs :
  - Image
  - Titre du poste
  - Description complète
  - Missions
  - Profil recherché
  - Conditions
  - Date limite
  - Zone géographique ciblée
  - Type d'action : Postuler / Participer / Sponsoriser
- Bouton **"Publier"** *(soumis à validation admin)*

---

### C4. Mes Offres Publiées
- Liste des offres publiées par l'entreprise
- Chaque offre affiche :
  - Titre
  - Date de publication
  - Date limite
  - Statut : *En attente / Validée / Expirée*
  - Nombre de candidatures reçues
- Actions : Modifier / Supprimer

---

### C5. Candidatures Reçues
- Liste des candidatures pour chaque offre
- Chaque candidature affiche *(accès gratuit)* :
  - Intitulé du profil
  - Sexe
  - Compétences / qualités
  - Bouton **"Programmer un entretien"**
  - Reste flouté *(photo, expériences, diplômes)*
- Chaque candidature affiche *(accès premium)* :
  - Photo de profil
  - Photos plein pied
  - Expériences détaillées
  - Diplôme et établissement
  - Bouton **"Appeler / Contacter"**
- Actions : Programmer / Rejeter

---

### C6. Parcourir les Candidats par Zone
- Carte / liste des candidats disponibles dans la zone
- Filtres : préfecture / compétence / disponibilité
- Chaque candidat affiche *(accès gratuit)* :
  - Intitulé du profil
  - Sexe
  - Compétences
  - Bouton **"Contacter"**
  - Reste flouté selon règles admin
- Chaque candidat affiche *(accès premium)* :
  - Infos complètes débloquées

---

### C7. Programmer un Rendez-vous
- Sélection du candidat
- Date et heure
- Localisation du lieu *(carte + adresse)*
- Notes complémentaires
- Bouton **"Confirmer le rendez-vous"**
- Notification automatique envoyée au candidat

---

### C8. Mes Rendez-vous
- Liste des rendez-vous programmés
- Chaque rendez-vous affiche :
  - Nom du candidat
  - Date, heure, lieu
  - Notes
  - Statut : *À venir / Passé / Annulé*
- Actions : Modifier / Annuler

---

### C9. Événements — Entreprise
- Même structure que la page événements candidat
- Possibilité de créer un événement avec type d'action : Postuler / Participer / Sponsoriser

---

### C10. Wallet & Abonnements — Entreprise
- Même structure que le wallet candidat
- Plans spécifiques entreprise affichés *(définis par l'admin)* :
  - Déblocage profil complet candidat
  - Accès photos, expériences, diplômes
  - Notifications prioritaires

---

### C11. Mon Profil Entreprise
- Logo, nom, secteur
- Description
- Localisation
- Badge de vérification *(si validé par l'admin)*
- Statut de viabilité / crédibilité
- Bouton **"Modifier mon profil"**
- Bouton **"Supprimer mon compte"**
